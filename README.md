# ReactJS calculator

A simple react js calculator.

![Screenshot](./calc.png)

## How to run / build 
Installing the packages
```bash
yarn  install
```
or
```bash
npm  install
```

Starting the development server
```bash
yarn start
```
or
```bash
npm start
```

Building the app
```bash
yarn build
```
or
```bash
npm run-script build
```
